# Winter Study 2026

방학 동안 서버, SQL, QA, 데이터 전처리 기초를 학습하며
실습 코드와 학습 기록을 남기는 개인 저장소입니다.

## Goals
- 서버 요청 → 처리 → 응답 흐름 이해
- 로그 기반 문제 분석 연습
- 재현 가능한 테스트 케이스 작성
- Pandas를 활용한 데이터 필터링 및 집계

## Topics
- FastAPI (API 서버, 예외 처리, 로깅)
- SQL (SELECT, JOIN, GROUP BY)
- QA (Test Case, Bug Report)
- Pandas (CSV 처리, 기본 통계)

## Structure
- /fastapi : FastAPI 실습 코드
- /sql : SQL 쿼리 연습
- /qa : 테스트 케이스 정리
- /pandas : 데이터 전처리 연습
